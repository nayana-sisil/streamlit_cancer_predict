BREAST CANCER DIAGNOSIS ASSISTANT

This is a machine learning web application. It predicts if a breast mass is benign or malignant based on cytology measurements.

LIVE DEMO
https://huggingface.co/spaces/nayanasisil2700/breast-cancer-diagnosis

FEATURES
- Interactive Sidebar to adjust values
- Radar Chart for visual analysis
- Real time Predictions
- Logistic Regression Model

TECHNOLOGIES
- Python
- Streamlit
- Scikit Learn
- Plotly
- Pandas

HOW TO RUN
1 Clone the repository
2 Install dependencies using pip install -r requirements.txt
3 Run the app using streamlit run app.py
